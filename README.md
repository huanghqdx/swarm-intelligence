# 幻灯片模板
## 使用说明
1. 下载并安装 TeXLive；
2. 下载本模板文件（使用 ```git``` 命令或在
[这里下载](https://github.com/K-JW/CCNU_BeamerTemplate/releases)）；
3. 编写 topic目录下相应的```.tex``` 文件；
4. 图片文件放置在pic目录下，命名格式为：主题+编号，如fish1.png
5. 使用 ```XeLaTeX``` 编译
